# GitHub Actions — Reusable Workflows POC

Documentation and presentation materials for our **GitHub Actions Reusable Workflows** proof of concept — demonstrating hands-on GHA expertise through a working centralized CI/CD platform.

## Structure

The content is organized in two parts:

- **Part 1 — Overview:** High-level capabilities, what we built, and why it matters
- **Part 2 — Deep Dive:** Architecture details, advanced patterns, and technical reference
